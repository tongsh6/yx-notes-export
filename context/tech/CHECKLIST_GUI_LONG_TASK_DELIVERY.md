# Checklist: GUI 长任务交付

## 交互可见性
- [ ] 有实时活动状态（不只进度条）
- [ ] 有无进度 watchdog（含最后状态）
- [ ] 长等待提示包含可操作建议

## 可控性
- [ ] 支持停止且文案说明生效时机
- [ ] 支持跳过当前（如业务允许）
- [ ] 停止/跳过后 UI 可收敛到明确终态

## 线程与稳定性
- [ ] 不使用 `terminate()` 强杀线程
- [ ] 阻塞调用返回后有 abort/skip 二次检查
- [ ] 结构化日志包含关键事件（retry/stall/stop/skip）

## 布局与可读性
- [ ] 中文文案无遮挡、无截断
- [ ] 按钮对齐一致，窄窗口无严重变形
- [ ] 深浅主题视觉节奏一致（高度、间距、边距）

## 交付验证
- [ ] 修改文件 `lsp_diagnostics` 无 error
- [ ] `pytest` 通过（至少相关 + 全量）
- [ ] 关键路径手测通过（连接、导出、卡顿、停止、跳过）
